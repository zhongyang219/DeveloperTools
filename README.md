# DeveloperTools
一些小工具集合。

使用 Visual Studio 2019 + Qt VS Tools 开发，Qt 版本：5.15.2。

界面框架使用QtitanRibbon。支持模块扩展。功能尚未完善，目前实现了删除代码注释的功能。
